Multi-agent legacy code migration — end to end.
Most "AI code conversion" tools are demos. CodeHarbor is a production platform that actually migrates enterprise legacy code at scale. Built on LangGraph, it orchestrates specialized agents that parse, analyse, convert, and validate code across stacks — with a VS Code extension so engineers can accept/reject changes inline.

DataHarbor
Enterprise data lineage — built on graphs.
DataHarbor answers the hardest question in enterprise data: "where did this data come from, and what depends on it?" Built a Neo4j-powered lineage platform that captures OpenLineage events, models them as a graph, and makes cross-job dependency exploration instant.
